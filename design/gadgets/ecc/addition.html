We will use formulae for curve arithmetic using affine coordinates on short Weierstrass curves, derived from section 4.1 of Hüseyin Hışıl’s thesis.

Incomplete addition

  • Inputs: P=(xp​,yp​),Q=(xq​,yq​)
  • Output: R=P⸭Q=(xr​,yr​)

The formulae from Hışıl’s thesis are:

  • x3​=(x1​−x2​y1​−y2​​)2−x1​−x2​
  • y3​=x1​−x2​y1​−y2​​⋅(x1​−x3​)−y1​.

Rename (x1​,y1​) to (xq​,yq​), (x2​,y2​) to (xp​,yp​), and (x3​,y3​) to (xr​,yr​), giving

  • xr​=(xq​−xp​yq​−yp​​)2−xq​−xp​
  • yr​=xq​−xp​yq​−yp​​⋅(xq​−xr​)−yq​

which is equivalent to

  • xr​+xq​+xp​=(xp​−xq​yp​−yq​​)2
  • yr​+yq​=xp​−xq​yp​−yq​​⋅(xq​−xr​).

Assuming xp​=xq​, we have

and​⟺⟺⟺⟺​xr​+xq​+xp​(xr​+xq​+xp​)⋅(xp​−xq​)2(xr​+xq​+xp​)⋅(xp​−xq​)2−(yp​−yq​)2yr​+yq​(yr​+yq​)⋅(xp​−xq​)(yr​+yq​)⋅(xp​−xq​)−(yp​−yq​)⋅(xq​−xr​)​======​(xp​−xq​yp​−yq​​)2(yp​−yq​)20xp​−xq​yp​−yq​​⋅(xq​−xr​)(yp​−yq​)⋅(xq​−xr​)0.​

So we get the constraints:

  • (xr​+xq​+xp​)⋅(xp​−xq​)2−(yp​−yq​)2=0
    • Note that this constraint is unsatisfiable for P⸭(−P) (when P=O), and so cannot be used with arbitrary inputs.
  • (yr​+yq​)⋅(xp​−xq​)−(yp​−yq​)⋅(xq​−xr​)=0.
