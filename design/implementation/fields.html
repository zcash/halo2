The halo2 Book

Fields

The Pasta curves that we use in halo2 are designed to be highly 2-adic, meaning that a large 2S multiplicative subgroup exists in each field. That is, we can write p−1≡2S⋅T with T odd. For both Pallas and Vesta, S=32; this helps to simplify the field implementations.

Sarkar square-root algorithm (table-based variant)

We use a technique from Sarkar2020 to compute square roots in halo2. The intuition behind the algorithm is that we can split the task into computing square roots in each multiplicative subgroup.
