fn configure( meta: &mut ConstraintSystem<F>, running_sum: Column<Advice>, table_idx: TableColumn, ) -> Self
where Self: Sized,

The running_sum advice column breaks the field element into K-bit words. It is used to construct the input expression to the lookup argument.

The table_idx fixed column contains values from [0..2^K). Looking up a value in table_idx constrains it to be within this range. The table can be loaded outside this helper.

§Side-effects

Both the running_sum and constants columns will be equality-enabled.

fn range_check( &self, region: &mut Region<'_, F>, element: AssignedCell<F, F>, num_words: usize, strict: bool, ) -> Result<RunningSum<F>, Error>

If strict is set to “true”, the field element must fit into num_words * K bits. In other words, the the final cumulative sum z_{num_words} must be zero.

If strict is set to “false”, the final z_{num_words} is not constrained.

element must have been assigned to self.running_sum at offset 0.
